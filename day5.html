html introduction

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. It defines the structure of a webpage by using a series of elements, tags, and attributes to organize text, images, links, and other multimedia elements.

css introduction

It allows you to apply styles to HTML documents by prescribing colors, fonts, spacing, and positioning.

bootstrap

Bootstrap is a popular front-end framework for building responsive and mobile-first websites. It provides pre-designed CSS, JavaScript components, and utility classes to quickly create modern and consistent user interfaces. It Includes pre-built responsive grid systems for mobile-first design.
